행정안전부_안전정보 통합공개 조회 서비스

국가안전정보 통합공개시스템은 재난관리책임기관에서 공개하는 시설 등에 대한 각종 안전점검․진단 등의 결과를 한곳에 모아 통합적으로 공개하는 시스템입니다. 시스템 내 보유중인 시설물 기본정보 및 안전정보를 openAPI로 제공합니다. 주요 안전점검정보는 전기, 승강기, 시설물안전, 식품판매시설, 어린이집, 어린이놀이시설, 병원시설, 호텔, 휴양림, 학교 등이 있으며 데이터 제공기관은 행정안전부, 문화체육부, 환경부, 농림축산식품부, 보건복지부, 해양수산부, 국토교통부, 중소벤처기업부 등입니다
