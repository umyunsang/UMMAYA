(https://www.reb.or.kr/r-one/portal/openapi/openApiActKeyPage.do)
통계표코드 확인
R-ONE → Open API → 통계코드 검색
(https://www.reb.or.kr/r-one/portal/openapi/openApiGuideCdPage.do)
조회를 원하는 ‘통계표코드’ 확인
서비스 통계목록 조회
(필수) 위에서 찾은 통계표ID를 요청인자(STATBL_ID) 값에 입력하여 ‘서비스 통계목록’ 조회
통계 세부항목 목록 조회
(STATBL_ID) 값에 입력하여 ‘통계 세부항목 목록’ 조회
(선택) 필요에 따라 요청인자(ITM_TAG)로 입력
그룹/분류/항목(‘통계코드 검색’에서도 확인 가능)
각 요청인자(ITM_TAG)에 해당하는 항목ID(ITM_ID) 및 항목명(ITM_NM)은 아래 URL에서 조회 가능
https://www.reb.or.kr/r-one/portal/openapi/openApiGuideCdPage.do
통계 조회 조건 설정 조회
(필수) 통계표ID를 요청인자(STATBL_ID) 값에 입력
(필수) ‘서비스 통계목록’에서 조회한 주기코드를 요청인자(DTACYCLE_CD) 값에 입력
(선택) 필요에 따라 요청인자(WRTTIME_IDTFR_ID) 값 입력
주기코드(DTACYCLE_CD) 값에 따라, 자료작성 시점(WRTTIME_IDTFR_ID) 입력 값 상이
(선택) 필요에 따라 요청인자(GRP_ID/CLS_ID/ITM_ID) 값 입력
각 요청인자에 해당하는 ID는 아래 URL에서 조회 가능
https://www.reb.or.kr/r-one/portal/openapi/openApiGuideCdPage.do
(선택) 필요에 따라 요청인자(START_WRTTIME/END_WRTTIME) 값 입력
주기코드(DTACYCLE_CD) 값에 따라, 자료작성 시점 시작일/종료일 입력 값 상이
