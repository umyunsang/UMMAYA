현재환율 API

조회수 305148

한국수출입은행이 제공하는 환율정보입니다.

■ 공지사항
- 기존 도메인 병행 가동 종료 예정입니다.(2026.4.30) 아래 개발 명세를 참고하시어 신규 도메인(oapi.koreaexim.go.kr)으로 변경하시기 바랍니다.
- 당행 환율 API에서 제공하는 데이터는 일환율 데이터로, 영업일 11시 전후로 업데이트 됩니다.

■ 개인정보보유기간 만료에 따른 인증키 파기 처리 관련
RESULT 3으로 반환되는 경우, 최초 신청 시 입력해주셨던 개인정보의 보유기간이 만료되어 API 키가 파기되었을 가능성이 높습니다.
개인정보보호법에 따라 보유기간(2년)이 지난 사용자의 경우 개인정보 및 사용정보 등이 파기 처리 됩니다.
파기 전 등록된 이메일로 '개인정보 수집 재동의' 안내 메일이 발송되며, 재동의 시 2년 연장됩니다.
파기된 키는 재사용이 불가하므로 신규 발급신청하시기 바랍니다.

※ 변경 이력
- 요청URL(Request URL) 도메인 변경에 따른 기존 도메인(www.koreaexim.go.kr) 병행 가동 중단 / 적용일시 : 2026.4.30(목)
- 요청URL(Request URL) 도메인 변경(www.koreaexim.go.kr -> oapi.koreaexim.go.kr) / 적용일시 : 2025.6.25(수)
- 일일 호출 가능 횟수를 1000회로 제한 (1000회 이상 호출 시 result:4 반환, 데이터 미제공) / 적용일시 : 2020. 7. 13(월) 오후 6시

※ 방화벽 해제 시 아래 내용 참고 바랍니다.
