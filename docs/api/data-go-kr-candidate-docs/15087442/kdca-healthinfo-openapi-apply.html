    OpenAPI 신청

    개인정보 수집·이용 동의(필수)

    질병관리청의 국가건강정보포털은 사용자의 민원사무 처리를 위해 개인정보를 수집합니다.

    [개인정보 수집·이용 목적] 민원사무 처리(보건교육자료실, 고객 문의, 건강정보(신규) 신청, 건강정보신청)
    [개인정보 수집 항목] (필수) 성명, 연락처(휴대전화번호), 이메일주소
    [개인정보 보유 및 이용기간] 2년(2년 후 재동의 필요)

    개인정보 수집에 동의하지 않을 수 있으며, 동의하지 않는 경우 민원사무 처리가 불가능합니다.
