❯ 모바일 신분증으로 본인인증 시뮬레이션 해줘

⏺ 모바일 신분증 본인인증 시뮬레이션을 위해 적합한 어탭터를 찾았습니다. 모바일 신분증(mDL)용 Verify ceremony를 제공하는 mock_verify_mobile_id 어탭터를
  사용하겠습니다.

  모바일 신분증 본인인증 시뮬레이션을 시도했지만, 현재 시스템에 해당 기능을 지원하는 어탭터가 등록되어 있지 않은 것으로 보입니다.

  제가 시도한 접근 방식:
  1. mock_verify_mobile_id - 모바일 신분증 전용 Verify ceremony
  2. mock_verify_module_modid - 모바일ID 모듈 Verify ceremony
  3. verify - 일반 인증 위임 토큰 발급
  4. mobile_id - 모바일 ID 인증
  5. mock_verify_module_simple_auth - 간편인증 모듈

  모두 "등록된 어탭터가 없음" 오류를 반환했습니다.

  현재 상황 분석


  이 문제는 다음과 같은 이유로 발생할 수 있습니다:
  - 테스트/모의 환경과 실제 운영 환경의 불일치
  - 해당 어탭터가 아직 시스템에 등록되지 않음
  - BM25 검색 결과와 실제 등록 상태 간 차이

  대안 제안


  1. 시스템 관리자에게 문의: 해당 어탭터들의 실제 등록 상태 확인
  2. 다른 인증 방법 사용: 간편인증(PASS, 카카오, 네이버, 토스) 또는 공인인증서 등으로 시뮬레이션 진행
  3. 실제 모바일 신분증 앱 활용: 실제 모바일 운전면허증 앱으로 본인인증 절차 체험

  현재 가능한 작업


  시스템에 실제로 등록된 다른 어탭터가 있다면, 해당 도구들을 통해 다른 유형의 본인인증 시뮬레이션을 진행할 수 있습니다. 예를 들어:
  - 간편인증 모듈을 통한 인증 시뮬레이션
  - 공인인증서 검증
  - 기타 등록된 인증 수단

  어떤 다른 유형의 인증 시뮬레이션이 필요하신가요? 또는 다른 공공 서비스 관련 작업이 있으신가요?
